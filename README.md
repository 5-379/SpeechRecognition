# SpeechRecognition

语音识别是人工智能领域的一个重要分支，近年来，语音识别在生活中的应用越来越广泛，例如语音助手、语音翻译、歌曲识别等等。

该项目计划实现一个中文语音识别系统，系统接收一段中文语音输入，输出其对应的文本信息。

## 对课题的理解

### 总体思路

- 查阅相关资料文档，学习相关知识
- 收集数据集
- 处理输入数据(将输入的语音转为频谱图)
- 构建语音模型(Speech Model)(CNN + LSTM/GRU + CTC)
    - Input： 声学特征MFCC
    - Output： 汉语拼音
- 构建语言模型(Language Model)
    基于概率统计的最大熵隐马尔可夫模型
    - Input： 汉语拼音序列
    - Output： 对应的汉字文本

## 项目的系统流程

1. 特征提取
2. 声学/语音模型
3. CTC 解码
4. 语言模型
